# Changelog

All notable changes to the AI ROI Dataset will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-02

### Added
- Initial public release of AI ROI Dataset
- 200 B2B AI deployment projects (2022-2025)
- 20 variables documented
- Full technical report (25-30 pages)
- Reproducible Jupyter notebook with 6 visualizations
- Data dictionary with detailed variable definitions
- README with quick start guide
- CITATION.cff for standardized citations
- LICENSE (CC BY 4.0)
- requirements.txt for Python dependencies

### Dataset Statistics (v1.0)
- **Total projects:** 200
- **Success rate:** 82.5%
- **Median ROI:** 159.8%
- **Sectors covered:** 10
- **Company sizes:** 3 categories (PME, ETI, Large)
- **Time period:** 2022 Q1 - 2025 Q4

### Documentation
- Technical Report: comprehensive methodology, results, limitations
- Jupyter Notebook: reproducible analysis with statistical tests
- Data Dictionary: all 20 variables fully defined
- README: installation, usage, citation guidelines

### Known Limitations (v1.0)
- Geographic scope limited to France (primarily Lyon/Auvergne-Rh√¥ne-Alpes)
- Selection bias: reflects successful deployment methodology
- Temporal heterogeneity: 2022-2025 spans rapid GenAI evolution
- Small sample sizes in Construction (n=5) and Energy (n=10) sectors

### Future Versions (Planned)
- v1.1 (Q2 2026): Add 50 additional projects from 2025
- v2.0 (2027): Longitudinal tracking (Years 3-5 ROI)
- v2.1 (2027): International expansion (Germany, UK data)

## Versioning Scheme

**MAJOR.MINOR.PATCH**

- **MAJOR:** Significant changes to dataset structure or methodology
- **MINOR:** Addition of new projects or variables
- **PATCH:** Corrections, documentation updates, bug fixes

## How to Report Issues

If you discover data quality issues, errors, or have suggestions:

1. **GitHub Issues:** [github.com/denisatlan/ai-roi-dataset/issues](https://github.com/denisatlan/ai-roi-dataset/issues)
2. **Email:** denis@denisatlan.fr
3. **Expected response time:** 5-7 business days

When reporting issues, please include:
- Version number (e.g., v1.0.0)
- Specific project_id or variable affected
- Description of the issue
- Expected vs. observed values (if applicable)
